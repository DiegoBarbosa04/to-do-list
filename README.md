# To Do List

Este é um projeto de To-Do List desenvolvido com ReactJS, Tailwind CSS e JavaScript.


---

## Descrição

Um aplicativo simples de To-Do List que permite:
- Adicionar tarefas
- Marcar tarefas como concluídas
- Visualizar tarefas pendentes e realizadas
- Remover tarefas
